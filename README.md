# sysmon — плавающий системный монитор (macOS и Windows)

Маленькое полупрозрачное окошко поверх всех окон. Каждая метрика — отдельный
**цветной блок** (своя плашка, цветная полоска и цвет текста). Набор, порядок и
цвета блоков настраиваются. Можно показывать любые из метрик:

- **NET** — скорость интернета (v загрузка, ^ отдача);
- **CPU** — загрузка процессора в %;
- **RAM** — память в %;
- **SWAP** — подкачка (объём и %);
- **DISK** — занято на системном диске, %;
- **IO** — скорость чтения/записи диска;
- **BAT** — батарея (% и «+» при зарядке);
- **FPS** — частота кадров / частота обновления экрана;
- **LOAD** — load average (только macOS/Linux);
- **PROC** — число процессов;
- **UP** — время работы системы;
- **TIME** — часы.

## Запуск

**macOS** — двойной клик по **run.command** (или `./.venv/bin/python monitor.py`).

**Windows** — двойной клик по **run.bat** (он сам поставит `psutil` и запустит).
Или из терминала:

```bat
pip install psutil
pythonw monitor_win.py
```

Нужен установленный [Python](https://www.python.org/downloads/) (галочку
«Add Python to PATH» при установке — включить). `tkinter` входит в Python,
ставить отдельно не нужно.

## Управление (одинаково на обеих ОС)

- **Перетаскивание** — тяни окошко мышкой за любое место.
- **Правый клик** по окошку — меню:
  - **Настройки…** — окно со списком всех метрик;
  - «Прозрачнее» / «Плотнее»;
  - «Выход».
- В окне **Настройки**:
  - слева цветная метка — цвет блока;
  - галочка — показывать метрику (добавить/убрать);
  - **▲ / ▼** — менять порядок блоков.
  - Изменения применяются сразу и сохраняются.
- Все настройки и позиция сохраняются в `config.json` рядом с программой.

## Настройка вручную (config.json)

```json
{
  "metrics": ["net", "cpu", "fps"],   // что показывать и в каком порядке
  "colors": { "net": "#34C759", ... }, // цвет каждого блока (HEX)
  "opacity": 0.9,                      // прозрачность окна (0.2–1.0)
  "font_size": 14,                     // размер шрифта
  "x": null,                           // позиция; null = угол по умолчанию
  "y": null
}
```

Доступные ключи для `metrics` и `colors`:
`net`, `cpu`, `ram`, `swap`, `disk`, `diskio`, `battery`, `fps`, `loadavg`, `procs`, `uptime`, `clock`.

Чтобы сменить цвет блока — поменяй HEX в `colors` (например `"fps": "#00E5FF"`).

## Как устроено

- **metrics.py** — общий кросс-платформенный код: реестр метрик, подсчёт через
  `psutil`, конфиг и цвета. Не зависит от ОС.
- **monitor.py** — macOS-версия. Окно `NSWindow` (PyObjC) на уровне
  `NSScreenSaverWindowLevel` (висит поверх всего, включая игры). FPS — честно
  через `CADisplayLink` (счёт кадров дисплея за секунду).
- **monitor_win.py** — Windows-версия на `tkinter`. Прозрачное окно поверх всех,
  блоки рисуются на `Canvas`. «FPS» — частота обновления экрана через Win32.

## Что можно добавить дальше

- Температуру / частоту CPU, GPU-загрузку.
- Мини-графики (спарклайны) вместо цифр.
- Автозапуск при входе в систему.
- Выбор цвета прямо в настройках (вместо правки JSON).
