# sysmon — плавающий системный монитор (macOS и Windows)

Маленькое полупрозрачное окошко поверх всех окон. Каждая метрика — отдельный
**цветной блок**, который можно перетаскивать и собирать в мозаику. Набор,
порядок и цвета блоков настраиваются. Доступные метрики:

- **NET** — скорость интернета (v загрузка, ^ отдача);
- **CPU** — загрузка процессора в %;
- **FRQ** — тактовая частота процессора;
- **RAM** — память в %;
- **MEM** — память в ГБ;
- **SWAP** — подкачка (объём и %);
- **DISK** — занято на системном диске, %;
- **FREE** — свободно на диске;
- **IO** — скорость чтения/записи диска;
- **BAT** — батарея (% и «+» при зарядке);
- **FPS** — частота, с которой рисуется окно/экран (60/120);
- **LOAD** — load average (только macOS/Linux);
- **PROC** — число процессов;
- **UP** — время работы системы;
- **TIME** — часы.

## Установка через pip

```bash
pip install sysmon-overlay
sysmon
```

Команда `sysmon` сама запускает нужную версию (macOS или Windows). Настройки
хранятся в `~/.config/sysmon-overlay/config.json`.

## Запуск из исходников

**macOS** — двойной клик по **run.command** (или `PYTHONPATH=src ./.venv/bin/python -m sysmon_overlay`).

**Windows** — двойной клик по **run.bat** (он сам поставит `psutil` и запустит).

Нужен установленный [Python](https://www.python.org/downloads/) (при установке на
Windows включи галочку «Add Python to PATH»). `tkinter` входит в Python.

## Управление

Общее: **перетаскивай пустое место окошка**, чтобы двигать весь оверлей по экрану.

### macOS

- **Иконка `▦` в строке меню** (вверху справа): показать/скрыть оверлей,
  настройки, выход.
- **Правый клик** по оверлею: **Настройки…** и **Скрыть**.
- **Редактор** (Настройки…) — одно окно, три зоны:
  - **Холст** — вся мозаика. Перетаскивай блоки — они **магнитно прилипают**
    друг к другу; кликом выбираешь; «Разложить заново» складывает столбиком.
  - **Инспектор** — про выбранный блок: описание, значение, **выбор цвета**
    (color picker), **мини-график** истории, «Убрать блок». Инспектор можно
    свернуть.
  - **Палитра** — все блоки карточками + **поиск**; клик добавляет/убирает.
  - Внизу: **прозрачность**, размер шрифта. Окно можно развернуть на **весь экран**.

### Windows

Правый клик по оверлею → простые настройки: галочки метрик и порядок ▲▼,
прозрачность, выход.

Все настройки и раскладка сохраняются в `config.json` рядом с программой.

## Настройка вручную (config.json)

```json
{
  "metrics": ["net", "cpu", "fps"],    // что показывать и в каком порядке
  "colors": { "net": "#34C759", ... },  // цвет каждого блока (HEX)
  "opacity": 0.9,                       // прозрачность окна (0.2–1.0)
  "font_size": 14,                      // размер шрифта
  "layout": { "cpu": [0, 0] },          // позиции блоков в мозаике (задаются в редакторе)
  "x": null, "y": null                  // позиция окна; null = угол по умолчанию
}
```

Ключи метрик: `net`, `cpu`, `cpufreq`, `ram`, `memused`, `swap`, `disk`,
`diskfree`, `diskio`, `battery`, `fps`, `loadavg`, `procs`, `uptime`, `clock`.

## Как устроено

Код пакета — в `src/sysmon_overlay/`:

- **metrics.py** — общий кросс-платформенный код: реестр метрик, подсчёт через
  `psutil`, конфиг, цвета, история значений. Не зависит от ОС.
- **monitor.py** — macOS-оверлей. `NSWindow` (PyObjC) на уровне
  `NSScreenSaverWindowLevel` (поверх всего, включая игры). FPS — честно через
  `CADisplayLink`. Иконка в строке меню через `NSStatusItem`.
- **editor_mac.py** — редактор мозаики для macOS (холст, палитра, инспектор,
  графики).
- **monitor_win.py** — Windows-версия на `tkinter`.

## Что можно добавить дальше
